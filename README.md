# BeautyHall - Салон краси

Красивий вебсайт для салону краси **BeautyHall**, розроблений з використанням сучасних технологій (HTML5, CSS3, JavaScript).

##  Особливості

- ✨ **Елегантний дизайн** - мякенька кольорова гама (персик, кремовий, золотий)
- 📱 **Адаптивний** - ідеально виглядає на всіх пристроях (мобільні, планшети, комп'ютери)
- 🎯 **Функціональний** - форма запису, портфоліо, контакти
- ⚡ **Швидкий** - оптимізований для швидкого завантаження
- 🔗 **Інтеграція WhatsApp** - можливість записатися прямо через мессенджер

## 📋 Розділи сайту

1. **Головна** - привабливий герой з інформацією про салон
2. **Послуги** - всі послуги салону:
   - 💅 Манікюр
   - 🦶 Педікюр
   - 💆 Масаж
   - ✂️ Жіноча стрижка
   - 💈 Чоловіча стрижка
   - ✨ Спеціальні пропозиції

3. **Про нас** - інформація про салон та власницю
4. **Портфоліо** - галерея робіт
5. **Запис** - форма для запису на послуги
6. **Контакти** - контактна інформація

## 🚀 Розгортання на Vercel

### Варіант 1: Git + Vercel Dashboard

1. Перейди на https://vercel.com
2. Натисни "New Project"
3. Вибери репозиторій `Hyperhen/Mother`
4. Натисни "Deploy"
5. Готово! Сайт буде доступний за посиланням

### Варіант 2: Vercel CLI

```bash
# Встанови Vercel CLI
npm i -g vercel

# Перейди в папку проекту
cd Mother

# Розгорни
vercel
```

## 💻 Локальне запущення

1. Клонуй репозиторій:
```bash
git clone https://github.com/Hyperhen/Mother.git
cd Mother
```

2. Відкрий `index.html` у браузері або використай Live Server
